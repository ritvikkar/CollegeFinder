College Search Webapp

About This Web Application

Created by Ritvik Kar, Anna Johnson, and Estelle Bayer for Software Design, Spring 2017.

This college finder uses 2014-2015 College Scorecard data, available for public use.

Use the sliders and check boxes provided to narrow your search criteria, or look up a college by name.

At most one specific demographic can be selected for any given search.

For more specific information about an individual college, click on its name to expand.
